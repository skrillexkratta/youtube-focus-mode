YouTube Focus Mode does not collect, store, or share personal data.

## Data Usage
This extension only uses Chrome storage to save user preferences such as:
- Focus Mode settings
- Work Mode timer state
- Streak and session counts

All data is stored locally in the user's browser using Chrome's storage API.

## No Data Collection or Tracking
This extension:
- Does NOT collect personal information
- Does NOT track browsing activity
- Does NOT use analytics or tracking scripts
- Does NOT sell or share user data

This extension does not transmit any data to external servers.

## Permissions
The extension only requests permissions required for functionality:
- storage (save settings)
- tabs (apply changes to the current tab)
- alarms (work mode timer)
